== Description ==

This plugin can solve the problem that Tibetan fonts are inconsistent with the font sizes on the front-end web pages when the default Latin font of WordPress is unmodified. Tibetan font are based on the Google font "Jomolhari".

此插件可以解决WordPress默认的拉丁字体未做改动的情况下，藏文前端网页字体大小不统一的问题，藏文字体采用谷歌字体“Jomolhari”。
